# IT Portfolio

A professional personal portfolio built with **Next.js 15**, **React**, and **Tailwind CSS**, featuring a minimalist design and a fully integrated contact system.

## 🚀 Getting Started

To start the development server:

```bash
npm run dev
```

The application will be available at [http://localhost:3000](http://localhost:3000).

## 📂 Project Structure

- `src/app/`: Core application routing and API logic.
- `src/components/`: Reusable UI components and page sections.
- `src/emails/`: React Email templates for the contact system.
- `public/`: Static assets and media.
- `docs/`: Technical guides and architecture documentation.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Email**: [Resend](https://resend.com/) & [React Email](https://react.email/)
- **Linting**: [Biome](https://biomejs.dev/)

## 📖 Documentation

Detailed technical documentation for specific systems (e.g., the contact form) can be found in the [docs/](/docs) directory.
